# Cat Fact

An android app which fetches cat facts from internet.

A simple Android app developed to demonstrate the following concepts.

1. Retrofit
2. Moshi
3. Coroutines
4. Data binding
5. Network connectivity
6. Recycler view
7. Flow
8. Paging
9. Coil
10. Binding adapters
11. Navigation graph
